# Grit Reports

A powerful, self-contained reporting and spreadsheet-analysis engine that runs
entirely in the browser. Upload an Excel/CSV file, configure grouping,
aggregation, calculated columns, filters, cross-tabs, and charts, and export a
formatted `.xlsx` — no server, no install, and your data never leaves the
browser.

The application lives in [`excel-group-analyze/`](./excel-group-analyze/) — see
its [README](./excel-group-analyze/README.md) for the full feature list and
quick-start instructions.
