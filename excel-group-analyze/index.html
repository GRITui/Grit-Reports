How everything works

Formulas (Calculated columns)

Reference columns in square brackets and combine with + - * / ( ) and numbers:

([Qty Ordered] - [Qty Shipped]) / [Qty Ordered] * 100

A formula can use any column checked in Data for calculation and any calculated column defined above it. Division by zero gives 0. Drag a column chip into the formula (or click it) instead of typing.

Aggregation per column

The small dropdown next to each data column picks how rows combine per group: sum, count, avg, min, max. Blank cells are excluded from avg/min/max.

Conditional aggregation (SUMIF / COUNTIF)

Section 4 builds one new column per block: choose a type (sum, count, count distinct, average, min, max), a value column (required for every type except Count — the dropdown starts blank so you must pick one), and one or more “and” conditions on any raw column — a row only counts toward the block's total when every condition passes, like Excel's SUMIFS/COUNTIFS. At least one condition must have a value filled in (or use “is blank”/“is not blank”, which need no value); a block with no active condition would just duplicate an unconditional metric, so Run Analysis rejects it. The result column works anywhere a data column does: calculated-column formulas, category rules, filters, sorting, cross-tab, and charts.

Date buckets

Check a date column in Group rows by and pick a granularity: day → 2026-07-14, week → 2026-W28 (ISO week), month → 2026-07. Works with real Excel dates and date text.

Category columns

Rules are checked top to bottom; the first match wins, otherwise the fallback label applies. Categories can be used in filters and sorting.

Lookup join

Left join: every main row keeps its data and gains the picked lookup columns from the first matching lookup row; no match leaves them blank. Join happens before grouping.

Cross-tab & charts

Cross-tab spreads one value column across a column's distinct values (max 40). Charts and the color scale read the final result table; the color scale is preview-only and not written to the .xlsx.

Chart extras (all preview-only): bars can be vertical or horizontal; Axis min/max pins the value axis range (blank = auto, ignored if min ≥ max); Unit shows ticks and tooltips as thousands (K), millions (M) or with a % suffix. Series by splits bars/lines into one colored series per value of a column (top 12 by total, the rest fold into “Other”), and Stacked stacks those series instead of placing them side by side.

Display formats

Per column: number (thousands separators), percent (a value of 87.5 shows as 87.5% and becomes a true Excel percent in the download), or text, each with a decimals choice.

Presets

Presets live in this browser. Use Export file / Import file to move them to another computer. Loading a preset onto a file that lacks some of its columns skips just those parts and warns you.
